### 3.1 竞品格局

| 竞品 | 核心能力 | 定价 | BAA差异化机会 |
|------|---------|------|-------------|
| **品茗"晓筑"** | BIM审图，规则引擎强 | ¥5000-20000/月 | BAA的AI主导闭环更智能，但品茗已有品牌信任 |
| **红瓦科技** | BIM翻模、族库 | ¥2000-5000/月 | 方向不同，非直接竞争 |
| **帮图科技** | CAD审图插件 | ¥1000-3000/月 | 直接竞品，价格低但功能弱 |
| **广联达** | 全生命周期BIM | ¥10000+/月 | 巨头但非聚焦审图 |
| **AI原生竞品（2026可能出现）** | 大模型直接审图 | 未知 | **最大威胁。** 如GPT-5直接支持图纸分析 |

### 3.2 竞争风险

**最大的竞品风险不是同行，是通用大模型能力的快速进化。** 如果2026-2027年GPT-5/DeepSeek-V4直接支持DWG图纸分析，BAA的"知识库壁垒"价值将大幅缩水。

**应对策略：**
- 速度优先：6-9个月内拿到真实付费客户，建立数据飞轮
- 深度优先：做通用大模型不会做的——"过程资料管理"+"全生命周期"这部分是通用AI的盲区
- 合规优先：获得住建部或地方审图中心的推荐/认证

## 六、论文证据对市场判断的补充支撑（2026-06-09）

> 以下引用基于arXiv及已发表论文检索结果，为市场评估提供第三方学术证据。

### 6.1 "通用AI模型威胁"风险重评

**原评估：** 通用AI模型能力进化速度被低估，是最大竞品威胁。

**论文证据：**
- **LLM for Building Regulation Interpretation**（Fuchs et al., 2024, arXiv:2407.19227）：LLM能理解规范语义，但**精确几何参数提取和条件推理仍有限**。即使是2024年最先进的LLM，在需要从图纸中精确提取尺寸、判断条件组合的场景下仍然不够可靠。
- **LLM Attribution Analysis for Code Compliance**（Shi et al., 2026, ICCCBE 2026）：小模型+领域微调可接近大模型水平，但即便如此，**LLM做精确合规判定需要归因分析**（解释判定依据），不能黑箱输出。
- **Fine-Tuning LLMs for Building Code QA**（Aqib et al., 2025, arXiv:2505.09686）：RAG+微调后准确率82.7%，仍有约**17%的错误率**。对于审图这种高责任场景，17%不可接受。

**修正：通用AI审图威胁在2-3年内被高估。**
- 纯LLM审图的准确率不足以替代人工审图（17%+错误率）
- LLM在精确几何参数提取上仍然不可靠
- 审图场景的法律责任要求"可解释的判定依据"，LLM的黑箱输出不满足
- **但3-5年后风险真实存在**——VLM+LLM融合模型正在快速发展（论文5+6）

### 6.2 市场验证数据的学术支撑

**论文证据：**
- **Exploring Gen-AI in Building Research**（Wan et al., 2024, arXiv:2410.01096）：综述论文指出AI审图、规范合规检查是**2024-2026年建筑AI最可能落地的场景**，学术和工业界都在关注。
- **LLM-Driven Code Compliance in BIM**（Madireddy et al., 2025, arXiv:2506.12814）：量化数据显示AI合规检查可**减少60%手动审查时间**。

**修正：市场方向正确，但竞争可能加速。**
- 学术界的活跃度意味着未来1-2年可能出现更多竞品
- BAA的窗口期大约6-12个月，需要快速拿到真实客户建立数据飞轮

### 6.3 差异化定位建议更新

**原建议：** 将产品定位从"AI审图工具"升级为"工程合规全流程管理系统"。

**论文证据支撑的修正建议：**
- 论文2+3+8的结论一致：**"LLM+规则引擎"混合方案**是当前学术界公认的可行路线
- BAA的差异化优势不应是"全流程管理"（通用AI也能做），而是**"精确可解释的合规判定"**
- 建议定位调整为：**"精确可解释的AI合规预检助手"**——不替代审图机构，而是帮设计师在报审前自检

## 七、核心建议

1. **立即启动客户验证。** 在开发MVP的同时，访谈10-20家设计院/审图单位，验证付费意愿和核心功能需求。不要等到产品做好才发现客户不买单。
2. **Y1收入目标下调至¥30-50万**，对应20-30家付费客户（¥199/月×12=¥2388/年），更符合早期产品市场匹配度现状。
3. **定价以"低门槛+高价值感"策略**，¥199/月起，先跑量再提价。
4. **警惕通用AI模型威胁，但不必过度恐慌。** 当前学术证据显示纯LLM审图在精确度上仍有较大差距。窗口期6-12个月。
5. **产品定位："精确可解释的AI合规预检助手"**，而非"全流程管理系统"。

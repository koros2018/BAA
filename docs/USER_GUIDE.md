# BAA 用户操作手册

> BAA (Blueprint AI Agent) — 建筑图纸AI合规审查系统
> 版本: v1.12.0

---

## 快速入门

### 启动服务

```bash
# 开发模式
python src/api/baa_api.py

# 生产模式（Docker）
docker compose up -d
```

打开浏览器访问 `http://localhost:8000/` 进入前端界面。

## 核心流程

### 1. 上传图纸

1. 进入「📐 图纸管理」页面
2. 点击选择 DXF 或 DWG 文件
3. 选择建筑类型：民用建筑 / 工业建筑
4. 可选勾选「YOLO图元检测」
5. 点击「上传并解析」
6. 等待解析完成，查看解析结果预览

> **支持的建筑类型:**
> - **民用建筑** (civil): 办公楼、住宅、学校、商场等
> - **工业建筑** (industrial): 厂房、仓库、电厂、数据中心等
>
> **格式限制:** 仅支持 .dxf 和 .dwg（最大50MB）
> **天正图纸:** 如果遇到 AC10xx 版本错误，请用 LibreCAD 另存为 DXF 后再上传

### 2. 执行审查

**方式一：上传后直接审查**
1. 上传解析后，在图纸列表点击「送审」按钮
2. 自动跳转到 AI 审图页面并开始审查

**方式二：批量送审**
1. 勾选多张图纸
2. 点击「批量送审」按钮
3. 查看批量审查结果汇总

**方式三：从 AI审图页选择**
1. 进入「🔎 AI审图」页面
2. 从下拉列表选择已解析图纸
3. 点击「开始审查」
4. 查看审查结果（违规列表支持筛选/搜索/分页）

### 3. 查看审查结果

审查结果包含：
- **概要卡片**: 违规总数、严重/主要/轻微分布
- **构件分布**: 图纸中包含的实体类型及数量
- **违规详情**: 每条违规的规范条款、实测值、要求值、严重等级
  - 🔴 严重 (critical): 偏差 > 30%
  - 🟠 主要 (major): 偏差 10%~30%
  - 🟡 轻微 (minor): 偏差 < 10%

### 4. 对比与修正

1. 进入「📋 对比演示」页面
2. 选择已审查的图纸
3. 查看：
   - **原始解析数据**: 图纸中提取的实体列表
   - **违规标注**: 逐条违规详情
   - **违规可视化叠加**: 实体在 Canvas 上的位置标注（按严重度着色）
   - **修正建议**: 自动生成的修正方案

### 5. 审查记录

1. 进入「📋 审查记录」页面
2. 所有审查结果自动保存（localStorage，最多50条）
3. 支持：
   - **搜索**: 按图纸名称、规范编号查找
   - **筛选**: 民用/工业/有违规/全部合规
   - **详情**: 点击卡片查看完整违规列表
   - **清空**: 一键清空所有记录

## API 密钥管理

### 创建密钥

1. 进入「🔑 密钥管理」页面
2. 系统自动获取管理令牌（无需手动配置）
3. 点击「+ 创建密钥」
4. 填写名称、选择权限等级、设置有效期
5. 复制生成的密钥值（⚠️ 仅显示一次！）

### 权限等级

| 等级 | 说明 |
|------|------|
| admin | 完全控制（创建/撤销密钥） |
| write | 上传图纸、执行审查、重构 |
| read | 查看健康状态、审查结果 |
| limited | 仅健康检查 |

### 连接配置

1. 进入「🔌 连接配置」页面
2. 配置后端服务地址（默认 `http://localhost:8000`）
3. 添加访问令牌（手动输入或从密钥管理导入）
4. 选择活动令牌用于 API 调用

## 常见问题

### Q: 上传 DWG 文件提示错误？
A: 确保是标准 DWG 格式（非天正T3加密）。天正图纸请用 LibreCAD 另存为 DXF 格式后上传。

### Q: 审查结果为空？
A: 可能原因：1) 图纸中未识别出有效实体；2) 实体类型不在原子函数目标范围内。尝试使用 YOLO 检测增强。

### Q: 如何重置管理员密码？
A: 停止服务，删除 `data/api_keys.json` 文件，重启服务后重新创建密钥。

### Q: 服务启动报端口占用？
A: 修改 `.env` 文件中的 `BAA_PORT=8000` 为其他端口。

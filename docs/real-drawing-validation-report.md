## 3. 关键发现与改进

### 3.1 已修复：EXIST-004 corridor误匹配 ✅

**问题**: EXIST-004（疏散指示标志）target_entities 包含 `corridor`，导致走廊实体被误判为"有走廊但没有疏散指示标志"，产生 390 个误报。

**修复**: 移除 corridor，target_entities 改为 `["exit_sign", "sign"]`。

**效果**: 总违规从 **1,539 → 414（降73%）**。基础+上部结构从 533→275，室外电气从 458→67。

### 3.2 待改进：door属性缺失

door 实体缺少 `width` 和 `fire_rating` 属性，导致：
- ATTR-001（防火门等级）: 实际值=0.0
- DIM-006（疏散门净宽）: 实际值=0.0
- DIM-009（疏散出口宽度）: 实际值=0.0

**建议**: 当关键属性缺失时，返回"无法判定"而非 0.0

### 3.3 待改进：走廊宽度精度

DIM-004 违规 99 项，实际值集中在 ~1.096m（阈值 1.1m），非常接近边界。可能是 ezdxf 维度提取的单位转换误差。

### 3.4 待改进：DWG解析覆盖率

5 张 DWG（IDC/云计算中心等）完全无法解析。建议：
- 安装 LibreCAD 后另存为 DXF 再试
- 或增加 ezdwg 对 T3 格式的支持
